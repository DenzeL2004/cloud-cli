# mws

CLI для управления локальными YAML-профилями в текущей директории.

## Что делает проект

Приложение поддерживает команды для работы с профилями:

- `profile create --name --user --project` создает файл профиля;
- `profile get --name` читает профиль по имени;
- `profile list` выводит список сохраненных профилей;
- `profile delete --name` удаляет профиль;
- `help` показывает доступные команды.

Профили хранятся в файлах `*.yaml` в текущей рабочей директории.  
Каждый файл содержит два строковых поля:

- `user`
- `project`

Имя профиля используется как имя файла.


## Сборка

```bash
go build
```

## Запуск

Пример запуска:

```bash
./mws profile create --name=test --user=example --project=new-project
```
